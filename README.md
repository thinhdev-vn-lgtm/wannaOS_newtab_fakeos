WannaOS Fake Extension
A FakeOS / Desktop Simulation extension inspired by Windows UI.

Features
- Desktop environment simulation
- Draggable windows
- Fake boot sequence
- Notification system
- App system (explorer, browser, decryptor)

Install
- Download ZIP
- Open Chrome/Firefox
- Load unpacked extension

You can also install this extension directly from the Firefox Add-ons Store if you are using Firefox.
